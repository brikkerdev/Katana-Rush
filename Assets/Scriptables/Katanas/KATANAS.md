## Creating New Katanas (AI Agent Guide)

New katanas are created via `katana_definitions.json` in this folder. Add entries to the JSON array, then in Unity click **Tools > Katana Factory > Import from JSON**.

### JSON Schema

```json
[
  {
    "name": "InternalName",
    "rarity": "Common|Rare|Epic|Legendary|Challenge",
    "name_en": "Display Name",
    "name_ru": "Russian Name",
    "description_en": "Ability description or - if none",
    "description_ru": "Russian description or - if none",
    "preset": {
      "baseSpeed": 20,
      "maxSpeed": 40,
      "speedAcceleration": 0.1,
      "laneSwitchSpeed": 10,
      "jumpForce": 24,
      "maxJumps": 0,
      "gravity": -50,
      "jumpDuration": 0,
      "jumpHeight": 0,
      "maxDashes": 3,
      "dashDuration": 0.4,
      "dashSpeedMultiplier": 3,
      "dashRegenTime": 1,
      "dashRegenDelay": 1,
      "dashInvincible": true,
      "dashDamage": 1
    },
    "iconPath": "Assets/path/to/icon.png",
    "modelPrefabPath": "Assets/path/to/model.prefab",
    "bladeMaterialPath": "Assets/path/to/blade.mat",
    "slashEffectPath": "Assets/path/to/slash.prefab",
    "trailEffectPath": "",
    "abilityPath": "",
    "challenge": null
  }
]
```

### Field Reference

| Field | Required | Notes |
|-------|----------|-------|
| `name` | Yes | Internal name, used for asset filenames (`Name_Katana.asset`, `Name_Preset.asset`) and localization keys (`katana_name_name`, `katana_description_name`) |
| `rarity` | Yes | One of: `Common`, `Rare`, `Epic`, `Legendary`, `Challenge` |
| `name_en` / `name_ru` | Yes | Display names for localization |
| `description_en` / `description_ru` | Yes | Ability description or `-` if no ability |
| `preset` | Yes | All movement/combat stats (see Preset Stats below) |
| `iconPath` | No | Unity path to sprite asset. Leave empty to assign later in Inspector |
| `modelPrefabPath` | No | Unity path to 3D model prefab. Leave empty to assign later |
| `bladeMaterialPath` | No | Unity path to blade material. Leave empty to assign later |
| `slashEffectPath` | No | Unity path to slash VFX prefab. Leave empty to assign later |
| `trailEffectPath` | No | Unity path to trail VFX prefab. Leave empty if none |
| `abilityPath` | No | Unity path to a KatanaAbility ScriptableObject asset. Leave empty if none |
| `challenge` | No | Set to `null` for non-Challenge katanas. See Challenge section below |

### Preset Stats

| Stat | Type | Default | Description |
|------|------|---------|-------------|
| `baseSpeed` | float | 20 | Starting run speed |
| `maxSpeed` | float | 40 | Maximum run speed |
| `speedAcceleration` | float | 0.1 | Speed increase per second |
| `laneSwitchSpeed` | float | 10 | Lane switching speed |
| `jumpForce` | float | 24 | Vertical jump velocity |
| `maxJumps` | int | 0 | Extra jumps (0 = single, 1 = double jump) |
| `gravity` | float | -50 | Gravity force |
| `jumpDuration` | float | 0 | Jump duration in seconds (0 = use force/gravity) |
| `jumpHeight` | float | 0 | Jump height in units (0 = use force/gravity) |
| `maxDashes` | int | 3 | Dash charges |
| `dashDuration` | float | 0.4 | Dash duration in seconds |
| `dashSpeedMultiplier` | float | 3 | Speed multiplier during dash |
| `dashRegenTime` | float | 1 | Seconds to regenerate one dash |
| `dashRegenDelay` | float | 1 | Delay before regen starts |
| `dashInvincible` | bool | true | Invincible during dash |
| `dashDamage` | float | 1 | Damage dealt when dashing into enemy |

### Challenge Katanas

For `Challenge` rarity katanas, provide a `challenge` object:

```json
"challenge": {
  "type": 3,
  "targetValue": 10000,
  "descriptionKey": "challenge_my_custom_key",
  "description_en": "Dash 10k times in total",
  "description_ru": "Russian translation"
}
```

Challenge type values: `0` None, `1` TotalDistance, `2` EnemiesKilled, `3` DashesUsed, `4` GamesPlayed, `5` CoinsCollected, `6` BulletsDeflected, `7` JumpsPerformed, `8` PlayTime, `9` HighScore, `10` LongestKillStreak, `11` CollectedFragments

### What Happens on Import

1. A `PlayerPreset` asset is created at `Assets/Scriptables/Katanas/{Rarity}/{Name}_Preset.asset`
2. A `Katana` asset is created at `Assets/Scriptables/Katanas/{Rarity}/{Name}_Katana.asset`
3. The katana is added to `KatanaDatabase.asset`
4. Localization entries are added to `LanguageDatabase.asset`
5. The katana ID is auto-generated (UUID)
6. Duplicate names are skipped
